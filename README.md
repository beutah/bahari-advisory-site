## 🌊 About Bahari

*Bahari* means **ocean** in Swahili — a nod to where this practice began. Bahari Advisory is a cloud and identity consulting firm founded by [Beutah Monyoncho](https://beutahmonyoncho.com), built on a decade inside financial markets before crossing into engineering.

We build secure, compliance-aware cloud, identity, and AI platforms for community banks, credit unions, and regulated financial institutions across the United States, with a long-term commitment to East Africa.

## 🛠️ What we build

- **DeskIQ** — AI-powered, compliance-aware member support on Amazon Bedrock
- **KYC Desk** — compliance onboarding automation
- **FinScan** — intelligent document extraction and financial tracking
- **IAM strategy** — vendor-agnostic identity across Okta, Entra ID, and Authentik

## 🎨 Design

Hand-coded in vanilla HTML, CSS, and JavaScript — no frameworks, no build step. Display type is [Fraunces](https://fonts.google.com/specimen/Fraunces), body text is Inter. The wave motif running through the site is a literal rendering of the brand's name.

**Palette:**
- Abyssal Navy `#0C2D4D`
- Seafoam Teal `#2BA39B`
- Teal Bright `#5EDED5`

## 📦 Stack

- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Hosting:** AWS S3 + CloudFront
- **DNS & SSL:** Cloudflare (Full strict mode)
- **Certificates:** AWS Certificate Manager

## 🚀 Running locally

```bash
git clone https://github.com/beutah/bahari-advisory-site.git
cd bahari-advisory-site
open index.html
```

No dependencies, no build.
